# Fiori/UI5 Fejlesztési Asszisztens

## Cél
SAP Fiori és SAPUI5 alkalmazások fejlesztésének támogatása, best practice-ek és modern megoldások.

## Prompt

```
Segíts SAP Fiori/UI5 alkalmazás fejlesztésében:

**Alkalmazás típusa:** [List Report, Worklist, Master-Detail, Custom]
**UI5 verzió:** [1.96, 1.108, 1.120+]
**Backend:** [OData V2, OData V4, RAP]
**Framework:** [Freestyle, Fiori Elements]

Segítség szükséges a következőkben:

1. **UI5 Komponens fejlesztés**
   - Controller logika
   - View (XML) struktúra
   - Model binding (OData, JSON, Resource)
   - Fragment-ek használata

2. **OData Service integráció**
   - Service binding
   - CRUD műveletek
   - Batch request-ek
   - Error handling
   - Mock server setup

3. **Best Practices**
   - MVC pattern betartása
   - Code reusability
   - Performance optimalizálás
   - Responsive design (sap.m)
   - Accessibility

4. **Modern UI5 Features**
   - Flexible Column Layout
   - Smart Controls
   - Expression Binding
   - Type definitions (TypeScript)

5. **Testing**
   - OPA5 tesztek
   - QUnit tesztek
   - Integration tesztek

**Specifikus feladat:** [IDE ÍRD LE A FELADATOT VAGY MÁSOLD BE A KÓDOT]
```

## Használati példa

**Input:**
```
Készíts egy Master-Detail Fiori appot, amely megjeleníti az értékesítési 
rendeléseket (SO) és a tételeket. OData V2, UI5 1.108.
```

**Output:** Teljes UI5 app struktúra Controller-rel, View-val, manifest.json-nal és OData binding-gal.
