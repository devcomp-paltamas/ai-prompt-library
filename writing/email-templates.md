# Email Template Creator

## Cél
Professzionális, hatékony email sablonok készítése különböző üzleti és személyes helyzetekhez.

## Prompt

```
Készíts professzionális email sablont a következő szituációhoz:

**Cél:** [pl. meeting egyeztetés, ügyfélfelvetel, panaszkezelés]
**Címzett:** [pl. új ügyfél, vezető, kolléga]
**Tónus:** [pl. formális, semi-formális, barátságos]

Az email tartalmazza:

1. **Tárgy sor**
   - Rövid és informatív
   - Cselekvésre ösztönző (ha releváns)

2. **Megnyitás**
   - Megfelelő megszólítás
   - Rövid kontextus

3. **Törzsszöveg**
   - Egyértelmű üzenet
   - Strukturált információk
   - Konkrét cselekvési lépések

4. **Lezárás**
   - Udvarias befejezés
   - Call-to-action (ha szükséges)
   - Megfelelő aláírás

5. **Opcionális elemek**
   - Follow-up időzítés
   - Mellékletek említése
   - További kapcsolatfelvételi opciók
```

## Használati példa

**Input:**
```
Cél: Meeting időpont egyeztetése potenciális ügyféllel
Címzett: Cégtulajdonos
Tónus: Professzionális, de barátságos
```

**Output:** Teljes email sablon tárgysorral, törzzsel és lezárással.

## Változatok

### Cold Email Outreach
```
Írj egy cold email-t:
- Új kapcsolat felépítésére
- Értékajánlat bemutatására
- Találkozó kérésére
- Követés stratégiával

[KONTEXTUS ÉS CÉL]
```

### Panaszkezelő Email
```
Készíts panaszkezelő email sablont:
- Empátiával
- Megoldás felaj ánlással
- Kompenzáció opcióval (ha releváns)
- Kapcsolat fenntartásával

[PANASZ TÍPUSA]
```
