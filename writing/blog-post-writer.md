# Blog Post Writer

## Cél
SEO-optimalizált, olvasóbarát blogbejegyzések írása strukturált formában.

## Prompt

```
Írj egy részletes, SEO-optimalizált blogbejegyzést a következő témában: [TÉMA]

**Célközönség:** [pl. kezdő webfejlesztők, vállalkozók, stb.]
**Hossz:** [pl. 1500-2000 szó]
**Tónus:** [pl. professzionális, barátságos, informális]

A blogbejegyzésnek tartalmaznia kell:

1. **Figyelemfeltő cím**
   - SEO-optimalizált
   - Érdeklődést keltő
   - Kulcsszó tartalmaz

2. **Bevezető bekezdés**
   - Hook az olvasó megragadásához
   - A bejegyzés értékének bemutatása
   - Rövid áttekintés a tartalomról

3. **Fő tartalom**
   - Logikusan strukturált alfejezetek (H2, H3)
   - Konkrét példák és esettanulmányok
   - Bullet pointok a könnyebb olvashatóságért
   - Releváns adatok és statisztikák

4. **Vizuális elemek javaslata**
   - Képek, infografikák leírása
   - Táblázatok tartalma

5. **Záró gondolatok**
   - Összefoglalás
   - Call-to-action
   - További olvasnivalók javaslata

6. **SEO elemek**
   - Meta leírás (155 karakter)
   - 5-7 releváns kulcsszó
   - Internal linking javaslatok
```

## Használati példa

**Input:**
```
Téma: AI használata kis vállalkozásokban
Célközönség: KKV tulajdonosok
Hossz: 1800 szó
Tónus: Barátságos, gyakorlatias
```

**Output:** Teljes blogbejegyzés strukturált formában, SEO elemekkel és publikációra kész tartalommal.

## Változatok

### Gyors Blog Draft
```
Írj egy rövid blog vázlatot [TÉMA] címmel:
- 3-5 fő pont
- Bevezető és lezárás
- Alapvető struktúra
- 500-800 szó
```

### List Article (Listicle)
```
Készíts egy "Top 10" vagy "X módja" típusú listicle-t [TÉMA] témában:
- Minden pont részletes kifejtése
- Praktikus példákkal
- Vizuális elemek javaslata
```

## Megjegyzések

- Mindig adj meg egyértelmű célt és célközönséget
- Az SEO kulcsszavakat előre kutatd ki
- A tónust igazítsd a brand voice-hoz
