# Social Media Manager

## Cél
Engaging social media tartalom tervezése és írása különböző platformokra.

## Prompt

```
Készíts social media tartalmat a következő paraméterek alapján:

**Platform:** [Facebook, Instagram, LinkedIn, Twitter/X, TikTok]
**Téma/Termék:** [IDE ÍRD A TÉMÁT]
**Cél:** [Engagement, brand awareness, konverzió, oktatás]
**Célközönség:** [Demográfiai és érdeklődési adatok]
**Tónus:** [Professzionális, casual, humoros, inspiráló]

A tartalom tartalmazza:

1. **Post szöveg**
   - Platform-specifikus hossz
   - Hook az első sorban
   - Releváns emoji-k (mértékkel)
   - Call-to-action

2. **Hashtag stratégia**
   - 5-10 releváns hashtag
   - Mix: nagy, közepes és kis reach
   - Branded hashtag-ek

3. **Vizuális elemek javaslata**
   - Kép/videó koncepció
   - Stílus és hangulat
   - Szöveg overlay javaslat

4. **Engagement stratégia**
   - Kérdések feltevése
   - Poll/szavazás ötletek
   - User-generated content felkérés

5. **Publikálási javaslat**
   - Optimális időpont
   - Gyakoriság
```

## Használati példa

**Input:**
```
Platform: Instagram
Téma: Új kávézó megnyitás
Cél: Brand awareness és követők növelése
Célközönség: 25-40 éves városlakók, specialty coffee rajongók
Tónus: Barátságos, közösségi
```

**Output:** Instagram post szöveg emoji-kkal, hashtag-ekkel, vizuális koncepcióval és engagement stratégiával.
