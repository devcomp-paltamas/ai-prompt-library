# Adat Elemző

## Cél
Adatok mély elemzése, minták felismerése és actionable betekintések generálása.

## Prompt

```
Elemezd a következő adatokat és adj átfogó elemzést:

**Adat típusa:** [CSV, táblázat, statisztika, stb.]
**Elemzési cél:** [trend azonosítás, predikció, összehasonlítás]
**Időkeret:** [napi, heti, havi, éves]

Az elemzésnek tartalmaznia kell:

1. **Leíró statisztika**
   - Átlag, medián, módusz
   - Szórás és tartomány
   - Outlierek azonosítása

2. **Trend analízis**
   - Növekedési/csökkenési mintázatok
   - Szezonalitás
   - Ciklikusság

3. **Korrelációk**
   - Változók közötti kapcsolatok
   - Oksági összefüggések
   - Befolyásoló tényezők

4. **Vizualizációs javaslatok**
   - Megfelelő diagram típusok
   - Dashboard elemek
   - KPI-k kiemelése

5. **Actionable betekintések**
   - Főbb megállapítások
   - Üzleti ajánlások
   - Következő lépések

[IDE MÁSOLD AZ ADATOKAT VAGY ADD MEG A KONTEXTUST]
```

## Használati példa

**Input:** Értékesítési adatok 12 hónapra

**Output:** Részletes elemzés trendekkel, seasonalitással, ajánlásokkal és vizualizációs javaslatokkal.

## Változatok

### Quick Data Summary
```
Adj gyors összefoglalót ezekről az adatokról:
- Főbb számok
- Legfontosabb trendek
- 3 kulcs betekintés

```

### Prediktív Elemzés
```
Az eddigi adatok alapján készíts előrejelzést:
- Várható trendek
- Növekedési/csökkenési előrejelzés
- Bizonytalansági tényezők

[TÖRTÉNELMI ADATOK]
```
